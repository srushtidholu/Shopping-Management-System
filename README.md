# Shopping-Management-System
## 1. Introduction
The Online Shopping Management System is a simple C++ based application that allows users to view products, add items to a cart, and generate a total bill. This system simulates the basic functionality of an e-commerce platform.
It is developed using Object-Oriented Programming (OOP) concepts such as classes and objects, making the system organized and easy to understand.

 ## 2. Objective
<li>To understand the concept of Object-Oriented Programming in C++ 
<li>To develop a simple shopping system 
<li>To allow users to select and purchase products  
<li>To calculate total cost automatically  
<li>To improve programming and logical skills

 ## 3. System Design
The system uses:
<li>Class for storing product data
<li>Class for managing cart data
<li>Functions for each operation (set product, display product, add to cart, show cart)

 ## 4.Conclusion
The Online Shopping Management System successfully demonstrates the basic functionality of an online shopping platform using C++. It helps in understanding how classes and objects work together to manage data and operations.
This project is simple yet effective in showing real-world application of programming concepts. It can be further improved by adding features like search, delete, file handling, and user authentication.
